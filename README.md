# digital-clock
digital clock with 7 segments display

For the 'clock' program - this program will show you in the cmputer screen the time. you will learn how to controll and use a RTC sensor.
For '7 SEGMENTS' file - this program will teach you how to display nubers on the 7 segments display. be caarefull to connects properly the jumper-wires! 
For '7 seg clock' file - the actual clock. this program combine the input fron the RTC and the output to the 7 segments display.

good luck!
